## 4. The PowerPoint‑to‑MP4 Pitfall  

A growing number of professionals export slide decks as MP4 for remote presentations. Unfortunately, a **2026 bug** in Microsoft 365 can cause the exported video to appear **pixelated**.

> *“I tried saving older files from 2025 as a MP4, and the same thing occurs, when previously, the files saved correctly.”* – *Microsoft Answers*  

### 4.1 What’s Going Wrong?  

- The export engine now defaults to a **lower resolution** (720p) unless you manually set a higher DPI.  
- Some custom fonts aren’t embedded, leading to blurry text rendering.

### 4.2 How to Fix It  

1. In PowerPoint, go to **File → Export → Create a Video**.  
2. Click **“Full HD (1080p)”** under *Resolution*.  
3. Under *Advanced Options*, set **“Use Presentation Quality”** for the video quality slider.  
4. If you still see fuzziness, **export as a PDF first**, then use a PDF‑to‑MP4 converter that respects vector graphics.
